# URVOTE Landing Project

URVote Merupakan aplikasi elektronik vote yang digunakan untuk proses pemilihan.

# Petunjuk Kontirubusi Pertama Kali

## Clone Repo ini

```cmd
git clone git@github.com:harissucipto/evoting-landing-page.git
```

- Buat Branch sesuai user name atau fiture yang ingin dikembangkan misalnya cipto

````cmd
git checkout -b namaBranchMu
``

- Push commit atatu branch local mu

```cmd
git add .
git commit -m "deskripsi yang kamu lakukan"
git push origin namaBranchMu
````

- Buka url repo ini lagi untuk melakukan pull request

```cmd
https://github.com/harissucipto/evoting-landing-page
```

lalu klik merge request di halaman githubnya lalu confirm sendiri
